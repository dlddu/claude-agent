| 상태 | 의미 |
|-----|------|
| ⏳ 대기 | 명세 작성 완료, 구현 대기 |
| 🚧 진행중 | 구현 진행 중 |
| 🧪 테스트중 | 구현 완료, 테스트 작성/실행 중 |
| ✅ 완료 | 구현 및 테스트 완료 |
| ❌ 실패 | 테스트 실패 또는 명세 불충족 |
| 🔄 변경필요 | 명세 변경으로 재작업 필요 |
